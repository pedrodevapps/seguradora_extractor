# Seguradora Extractor - Backend Python para OCR

## Como rodar localmente:
1. Crie um ambiente virtual (opcional)
2. Instale as dependências:
   pip install -r requirements.txt
3. Rode o servidor Flask:
   python main.py
4. Acesse http://localhost:5000/upload

## Como usar com Deta Space:
1. Instale a CLI: https://deta.space/docs/en/cli/install
2. Faça login:
   space login
3. Suba o app:
   space push

Pronto! Seu app estará online.
